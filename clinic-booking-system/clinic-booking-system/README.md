# Clinic Booking System

Automated WhatsApp booking and reception workflow for dental & medical clinics.

![Clinic workflow overview](./clinic-workflow.jpg)

## Overview
- Replaces manual receptionist chat with an AI assistant.
- Handles FAQs, booking, rescheduling, and reminder messages.
- Stores all inquiries and appointments in structured Google Sheets.

## Stack
- n8n (self-hosted on VPS)
- OpenAI (AI assistant replies)
- WhatsApp Business (clinic number)
- Google Sheets + Google Calendar

## Impact
- 90%+ of incoming messages handled automatically.
- Response time reduced from hours to under **2 minutes**.
- No missed inquiries, clear log of all patient conversations.
